# exzoom 是一个 jquery 放大镜插件

![DEMO](https://github.com/weihaipy/exzoom/raw/master/demo/demo.png "DEMO")

# 使用方法:
* 初始化: 
```javascript
$("#exzoom").exzoom();
```
* 初始化后可以设置大图:
 ```javascript
 $("#exzoom").exzoom("setImg", "http://www.jb51.net/images/logo.gif");
```
